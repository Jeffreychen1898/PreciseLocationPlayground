# Precise Location Playground
This repository is for testing and playing around with precise location for my MDE group.

# Visualize The Data
The data I collected are stored in text files like `points.txt` and `points2.txt`. To visualize
these points with matplotlib, run the following commands:

 - Create the virtual environment: `python3 -m venv env`
 - Install the libraries: `pip install -r requirements.txt`
 - Run the program: `python graphPoints.py`
